What is the difference between a block and an inline tag? Give examples of each one.
Block can't share line with another but inline can. Most of thags are block but some are inline such as img and link.

What is normal flow in HTML?
the way that Block and Inline elements are displayed on a page before any changes are made to their layout.

What does float do? Why do you need to clear the float?
position an element to the left or right of its containing element, allowing text and inline elements to wrap around it. Clearing the float ensures that the container encloses its floated children correctly.

What do absolute, fixed, and relative do?
absolute: position an element relative to its nearest positioned ancestor.
fixed: position an element relative to the browser window.
relative: position an element relative to its normal position.

How can you make nice navigation buttons for mobile devices?
For mobile device, you can use media query to change the layout of the page.
