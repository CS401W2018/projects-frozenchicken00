  1. Internet:

    A vast global network throughout the world to be able to communicate each other and obtain any information.

  2. Web:

    Web is a network of Internet servers that can handle content with specific formatting.

  3. How are the Internet and Web different?:

    Internet is more focusing on communicating with each other and Web is more focusing on sharing information.

  4. ASCII and Unicode:

    ASCII is a standard code to represent characters in US and Unicode is an international encoding standard.

  5. What is UTF-8:

    UTF-8 is a variable-width character encoding capable of encoding all 1,112,064 valid character

  6. URL:

    URL is a reference to a web resource that specifies its location on a computer network

  7. Domain Name:

    Domain Name is a human-readable name that is used to represent the IP address of a website.

  8. HTTP:

    HTTP is a protocol that allows the fetching of resources, such as HTML documents.

  9. HTTPS:

    HTTPS is a secure version of HTTP and it uses encryption for secure communication over a computer network

  10. IP Address:

    IP address is a unique address that identifies a device on the Internet or a local network.

  11. DNS:

    DNS is a system that translates domain names to IP addresses.

  12. W3C Recommendations:

    W3C Recommendations are the final stage of the standardization process by the World Wide Web Consortium and are considered Web standards.

  13. XML, XHTML, HTML, HTML5:

    • XML is a flexible, markup language used for storing and transporting data
    • XHTML is a stricter and cleaner version of HTML
    • HTML is a markup language used for creating web pages
    • HTML5 is the latest version of HTML
